Abstract

STOL is a web application that allows users to bypass the storage limits of regular drive services by allowing you to store files in the cloud while being connected on multiple drive accounts, without having to worry that uploading a certain file will exceed the limit of one of those particular accounts.

Introduction

This article will serve as a tutorial on how to use the application. All of the screenshots are from the desktop version, but accessing it from a mobile phone will not alter the interface in significant ways, so this guide is for all types of devices.

Landing Page

The landing page is very easy to use, the header provides relevant options for navigation according to authentication status and the page only has one other button which leads to the login screen.

Login and register pages

The user can easily fill out the forms to login/register and can press the forgot your password text to put in his email and receive a password reset email.

About us page

The about page contains information about the creators of the application and links to the documentation of the application. (on the bottom of the page)

Main Page

The basic layout of the page, the user has buttons on the right(under the search bar on mobile) that allow him to: upload files, create folders, connect drive accounts and access the dashboard(admins only)

When the + button is clicked, a modal will appear that prompts the user to upload files. The user can also drag and drop files from his system! If no drive accounts are connected, the user won't be able to upload files.

When the folder icon is clicked, the user is prompted to create a folder.

When the profile button is clicked, the user can see which drive services are connected on his account and, connect to the others!

When hovering over a file the user can choose to download or delete the file.(he needs to be connected to the drive accounts that have the fragments of the file)

Dashboard

The dashboard page enables admins to view user information such as storage space remaining and number of files, but they can also select table rows and export them in CSV format. Selected table rows are grey instead of black. Admins can also delete user accounts(non-admin ones) by clicking on the delete cell. A search bar is available if they want to search for a particular user by email.
