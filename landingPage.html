Welcome to STOL

What is STOL?

STOL is a smart and easy-to-use web application that allows users to store and manage large files, for free!

How is it done?

Behind the scenes, the application implements a very clever arhitecture that takes advantage of numerous cloud-storage services, such as Dropbox, Google Drive and Microsoft OneDrive.

Who created STOL?

The app was created by 3 students from the Faculty Of Computer Science in Iasi as their final project for the Web Technologies subject.
For more information please feel free to visit the About Us page.

How can I use STOL?

It's all very simple! To start using the application you can easily register or log in with an existing account.
After that, everything should be made very easy by our beginner-friendly user interface!
